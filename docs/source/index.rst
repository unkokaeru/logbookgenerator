Installation
------------

To install logbookgenerator, simply run:

.. code-block:: bash

    pip install logbookgenerator

Usage
-----

After installation, you can use logbookgenerator by
running:

.. code:: bash

    $ python3 -m logbookgenerator
    # or
    $ logbookgenerator
